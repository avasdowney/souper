# souper

A fun web-based arcade game inspired by the classic "Tapper" game, where you serve soup to hungry customers!

## How to Play

1. Open `index.html` in a web browser
2. Use the **A, S, D, F** keys to serve soup in lanes 1-4
3. Serve customers before their timer runs out
4. Earn 10 points for each customer served
5. Game ends if a customer's timer reaches 0 or they reach the right edge

## Game Features

- **4 horizontal lanes** where customers appear from the left
- **Timer system** - each customer has 30 seconds before they leave
- **Score tracking** - earn points for successful serves
- **Restart functionality** - play again after game over
- **Responsive design** - works on different screen sizes

## Technical Details

- Pure HTML, CSS, and JavaScript (no frameworks required)
- Uses emoji for characters and game elements
- Smooth animations and visual feedback
- Clean code with constants and proper resource management
